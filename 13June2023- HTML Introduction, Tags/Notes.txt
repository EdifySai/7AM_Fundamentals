HTML: 

Hypextext Mark up Language  is used to describe the static web 
pages on a browser.

We cannot create dynamic web pages using HTML , however we can use Javascript 
to develop it.

Mark up Language:

It is a tag based Language where the data / ui is represented by using 
tags. 

Tag: A tag is a keyword that is enclosed in angular brackets.

syntax:  <html>, <head>, <body>, <h1>.....

Every opening tag should have a closing tag, if the tag holds some 
content.

 <h1>Hello World </h1>

Q1. How does the browser display html pages on the UI?

Whenever we provide a html page to the browser, then browser creates 
a DOM (Document Object Model) tree.

The render engine picks up the content from the DOM tree and renders on the UI. 

If any updates happen to the DOM, then the render engine picks the latest content from DOM and displays it on the 
UI.
