HTML Lists:

Sometimes we want the data to be shown in the form of a list, then we 
can use HTML Lists. 

A list is a set of items.

There are 2 types of lists.

1. unordered :

tag: <ul>

<html>

<head>

</head>

<body>

    <ul style="list-style-type: disc">
        <li>Item 1</li>
        <ul>
            <li>Item 1.1</li>
            <li>Item 1.2</li>
            <li>Item 1.3</li>
            <li>Item 1.4</li>
            <li>Item 1.5</li>
        </ul>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
        <li>Item 5</li>
    </ul>

</body>

</html>


2. ordered

tag: <ol>

<html>

<head>

</head>

<body>

    <ol start="3" type="i">
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
        <li>Item 5</li>
    </ol>

</body>

</html>
